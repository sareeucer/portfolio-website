Hakkımda

Yeditepe Üniversitesi Hemşirelik öğrencisiyim ve aynı zamanda yazılım geliştirme alanında kendimi geliştiriyorum. Problem çözmeyi ve kod yazmayı seviyorum. Yaptığım her işe %100 katkı sağlamayı hedefliyorum. C#, SQL, HTML5, CSS, JavaScript, ASP.NET Framework (MVC, Web API, Identity), ASP.NET Core (Razor Pages, Middleware, Identity, Web API), WCF, Microsoft Azure ve Docker gibi teknolojiler üzerinde çalıştım. Gerçek dünya problemlerine çözüm sunan ve geniş kitlelere hitap eden yazılım projeleri geliştirmeye büyük bir tutku duyuyorum.

  • Diller: C#, JavaScript, HTML5, CSS, SQL
  • Veritabanları: Microsoft SQL Server, MySQL
  • Frameworkler: ASP.NET Framework (MVC, Web API, Identity), ASP.NET Core (Razor Pages, Middleware, Identity, Web API), WCF
  • Bulut & Platformlar: Microsoft Azure, Docker
  • Araçlar ve Teknolojiler: Git, Visual Studio, GitHub, Azure DevOps

Yazılım geliştiricisi olarak becerilerimi birleştirerek, mesleki gelişim, ilginç deneyimler ve kişisel gelişim sağlayan zorlu bir pozisyonda çalışma fırsatı arıyorum.

Deneyimlerim

Yazılım Geliştiricisi
  • Gelişmiş kullanıcı arayüzü ve admin paneline sahip, C# ve ASP.NET Core kullanılarak hazırlanmış bir restoran yönetim sistemi geliştirdim.
  • Hastane web sitesi, kahveci web sitesi ve mobilya mağazası sitesi olmak üzere 3 farklı responsive web projesi oluşturdum; bu sitelerde kullanıcı girişi, ürün listeleme, iletişim formları ve yönetici panelleri gibi özellikler geliştirdim.
  • Projelerde 20+ RESTful API geliştirdim; kullanıcı kayıt/giriş, ürün işlemleri, sipariş takibi ve içerik yönetimi gibi işlevler sağladım.
  • Kod yapısını sadeleştirerek ve veritabanı sorgularını optimize ederek performansı %25 oranında artırdım.
  • Kullanılan Teknolojiler: C#, ASP.NET Core, ASP.NET MVC, SQL Server, Razor Pages, HTML5, CSS, JavaScript, Entity Framework
Program Analisti
  • ASP.NET Core ve Entity Framework kullanarak restoranlar için sipariş ve masa yönetimi sağlayan modüler bir masaüstü ve web uygulaması geliştirdim; sistem çoklu kullanıcı desteğiyle farklı roller için (admin, garson, mutfak personeli) özelleştirildi.
  • Kahveci ve mobilya mağazası için geliştirilen responsive web sitelerinde ürün katalogları, kullanıcı yorumları ve sipariş sistemleri gibi fonksiyonlar yer aldı; tüm kullanıcı arayüzleri Razor Pages ve CSS ile modern şekilde tasarlandı.
  • Otomasyon sürecini hızlandırmak amacıyla yazılım mimarisini yeniden düzenledim; veritabanı işlemlerini optimize ederek sayfa yükleme ve işlem sürelerinde %30'a kadar iyileşme sağladım.
  • ASP.NET Identity ile güvenli kullanıcı kayıt/giriş sistemi entegre ettim; roller bazında yetkilendirme sistemi geliştirdim (örnek: kullanıcı, admin, satış temsilcisi).
  • Kullanılan Araç ve Teknolojiler: C#, ASP.NET Core, Razor Pages, Entity Framework, SQL Server, HTML5, CSS3, JavaScript, Docker, Azure
Yazılım Geliştirme
  • ASP.NET Core ve JavaScript kullanarak restoranlar için bulut tabanlı bir stok ve sipariş takip sistemi geliştirdim; ürünlerin stok durumu, fiyatları ve tedarik tarihleri sistemde anlık olarak izlenebiliyor.
  • Yönetici panelinde eşik değeri aşıldığında otomatik e-posta ve sistem içi bildirim gönderen bir uyarı sistemi oluşturdum; bu sayede stok eksikliği ya da yoğun sipariş durumları erkenden tespit edilebiliyor.
  • Kullanıcı dostu grafikler ve tablolamalar ile ürün hareketlerini görselleştirdim; bu özellikler, yöneticilerin karar alma süreçlerine yardımcı oldu.
  • Kullanılan Araçlar: C#, ASP.NET Core, JavaScript, Razor Pages, Entity Framework, SQL Server
Proje Stajyeri
  • Hastane yönetim sistemi için gereksinim toplama ve Entity-Relationship (ER) diyagramları hazırlama gibi analiz ve tasarım faaliyetlerini yürüttüm.
  • Bir restoran yönetim programı için, kullanıcı talepleri doğrultusunda veritabanı tasarımı ve veri akış diyagramları oluşturarak sistemin temel yapı taşlarını oluşturduk.
  • RESTful API geliştirme sürecinde, admin paneli ve müşteri sipariş takip sistemleri için fonksiyonel bir çözüm sundum; ürün listeleme, sipariş yönetimi ve kullanıcı yönetimi gibi işlemleri kolaylaştırdım.
  • Kullanılan Araçlar: C#, ASP.NET Core, SQL Server, Entity Framework, Razor Pages
